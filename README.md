# FinPickTeam_BE
**사회초년생을 위한 금융상품 추천 및 비교 플랫폼**

## 프로젝트 개요
**FinPick**은 금융 문맹을 줄이고, 사회초년생들이 쉽게 재테크를 시작할 수 있도록 도와주는 금융 플랫폼입니다.  
금융 상품의 추천 및 비교 기능, 자산 통합 조회, 챌린지 기반 금융습관 형성을 주요 기능으로 제공합니다.

## 목표
- 금융 지식이 부족한 사용자도 쉽게 접근할 수 있도록 UX/UI 및 콘텐츠 구성
- 사용자 맞춤형 금융상품(예/적금, 펀드, 주식) 추천
- 금융 데이터 기반 소비 분석 및 리포트 제공
- 챌린지를 통한 습관 형성 및 동기 부여
- 마이데이터 연동 기반 통합 자산 관리
